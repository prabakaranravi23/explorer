GIT Commands

To generate ssh key?

ssh-keygen-t ed25519 -C "your@gmail.com" - this is used to generate ssh key

git config --global user.email"your@email.com

It is used to identify the user email id

git config --global user.name"your_name"

It is used to identify the username

git clone

it is used to clone the repository in local machine

git status

it is used to check the status of local file

git add .

this is used to add many files without specifing the file name

git commit -am"message"

This is used to record changes in the code repository

git push

This is used to update the code or upload to code repository in github

git restore

This is used to restore the old file without any current updation

git fetch

This is used to fetch data about the branch details

git checkout

This is used to switch from main to branches
