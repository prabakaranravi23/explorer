GIT Commands

  1. To generate ssh key

    ssh-keygen-t ed25519 -c "your@gmail.com" - this is used to generate ssh key

  2. git confiq --global user.name"your_name"

    It is used to identify the username

    Example

    for example if you're name is prabakaran then the command is git confiq --global user.name "prabakaran"

  3. git confiq --global user.email"your@email.com"

    It is used to identify the user email id

    Example

    for example if you're email id is prabakaranaug93@gmail.com then the command is git confiq --global user.email "prabakaranaug93@gmail.com"

  4. git clone

    It is used to clone or copy of that respository at in a new directory, at another location.

    Example

    for example we have file in respository with url "exercise/e2/git_commands.html" if we want to create the same file in the current location we can use this command git clone exercise/e2/git_commands

  5. git status

    It is used to check the status of the local file in the current location

    Example

    for example the command to check the status is git status

  6. git add.

    this is used to add many files without specifying the file name

    Example

    for example if we have 2 or more files to add in the respository then the command is git add .

    git add"filename"

    this is used to add only the specified file to the repository

    Example

    for example if we have only 1file to add in the respository called "index.html" then the command is git add index.html

  7. git commit -am"message"

    this is used to record changes in the code respository

    Example

    for example if we have done staging for our file then the next step is to commit the file to the repository the command is git commit -am"new file is added" and always the commit should happen with message in it.

  8. git push

    this is used to update the code or upload the code repository in github

    Example

    for example after staging and commit then comes the push which is used to push the code to the repository the command is git push

  9. git restore

    this is used to restore the old file without any current updation

    Example

    for example if want to get the old file without the current updation then the command is git restore index.html

  10. git fetch

    this is used to fetch data about the branch details

    Example

    for example if we are having added new branch in our respository then the local machine find difficult to find it, if we use this command git fetch then it will be easy for the local machine to identify the new branch

  11. git checkout

    this is used to switch from main to branch or branch to main

    Example

    for example if we want to switch from main to other branch we can use this command git checkout index

  12. git pull

    this is used to download content from a remote repository and immediately update the local repository to match that content.

    Example

    for example after fetching the details of the branches then we have t pull the content to local repository git pull
